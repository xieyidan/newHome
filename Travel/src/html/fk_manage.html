会员与积分规则

概述

1. 会员是居家游对平台用户按照消费能力和对平台粘性划分的用户等级,各等级享有对应的服务及特权。划分4种会员等级:普通会员、银卡会员、金卡会员、钻石会员；

2. 积分是居家游针对会员消费、登录、分享等操作发放的平台内科折现的点击积分，每100积分可折现1元现金，可在平台内消费，不可提现；

3. 订单支付环节积分可抵现。

会员的等级机制

1. 会员的等级评定根据三项指标（消费金额、消费次数、积分）进行，消费作为积分的主要来源直接提现用户的消费能力，同时会员与平台的粘性越高获取到积分的途径越多；

2. 会员的等级评定分两个阶段执行

等级 消费金额 消费次数 积分 满足条件
普通 0 0 0 无
银卡 600 2 1200 任一
金卡 2400 5 3600 任一
钻石 6000 8 7800 任一

说明：成为银卡会员的条件是：消费达到600，消费次数2次，积分达到800，满足任一条件。

3. 会员的时效性，达到既定条件即刻升级为会员，会员等级为一年有效期，有效期到了之后会根据到期前一年内用户信息，按照当时的规则重新进行评定；

4. 获取积分途径： 1.消费：消费金额 x 消费获取积分系数 = 积分； 2.登录签到：每次获取积分 20，每天限一次； 3.分享：每次获取积分 50，每天限一次； 4.评论：每次获取积分 100； 5.完善资料：获取积分 200； 6.活动：参加居家游平台活动获取相应积分，具体以活动为准；
